Parking.ru объявляет Глобальное потепление

Блог компании Parking.ru
Приветствую тебя %username%.
Наша хостинг-компания Parking.ru уже более 10 лет занимается Windows хостингом и мы не могли пройти мимо такого знаменательного события, как 35-летие нашего партнера — компании Майкрософт. И поэтому мы приготовили сюрприз для всех кто хотел попробовать Windows, но по каким то причинам этого не делал.

И так мы объявляем "Глобальное потепление" — в рамках акции с 5 апреля вы можете получить виртуальную машину Hyper-V (512МБ) на Windows Web Server 2008 R2 всего за 100 р. в месяц!



Акция будет идти с 5 апреля по 30 мая. В акции может принять каждый, кто хочет попробовать перенести свой проект на Windows Server с других ОС.

Приняв участие вы получаете за 100 рублей в месяц VPS следующей конфигурации:
— Hyper-V RAM 512Мб, HDD 40Гб, CPU 10-50%, трафик неограничен
— Windows Web Server 2008 R2 (x64) + IIS 7.5 +.NET framework+PHP5 (FastCGI/Wincache) + MS SQL 2008 Express + MySQL5+WPI
— Все лицензии включены.

По окончании 6 месяцев, будет предложено сменить тариф на один из доступных в текущий момент, подробнее можете прочитать в нашем FAQ.
