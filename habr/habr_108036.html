[Перевод] Разделение JVM на платную/бесплатную версии: истерия, начавшаяся с поста в твиттере

Java *
В продолжение топика Oracle намеревается разделить JVM На платную/бесплатную версии

pelegri, редактор известнейшего блога о Java и иже — The Aquarium, отписался про ту истерии, которая началась после заявления представителя Oracle относительно Premium-версии JVM.

Мой вольный перевод со вставками необходимых ссылок ниже.


"
Это — история одного твита…

На прошлой неделе Адам на QCon в Сан-Франциско представил доклад «The Road Ahead for Java» относительно дальнейшего развития Java SE. В пиар-части доклада было:

Oracle JDK и Java Runtime Environment (JRE) будут доступны для свободного скачивания; никаких изменений в схеме лицензирования не предусматривается.

Платные услуги: JRockit Mission Control, JRockit Real Time, Java for Business и Enterprise Support продолжат предоставляться за отдельную плату.

Даже и не знаю, какие конкретно слова Адам использовал, но Michael Nygard запостил следующего содержание твит:

«В наши намерения входит предоставление Premium-версии JDK» — это было сказано к теме об опен-сорсной JDK. #qconsf

Ничего такого, просто твит. Может быть, так Адам и сказал. Проблема не в этом, а в произошедшем позднее.

Этот твит перетвитили столько раз, что в какой-то момент потеряли часть про опен-сорс, оставив часть про платную версию, и ещё какой только отсебятины не добавили. Всё это попало в руки журналистам, которые распространили «новость» по новостным сайтам, блогам и твиттеру. В-общем, всё произошло очень быстро: Майкл опубликовал твит в субботу 6-го числа, и вчера, погуглив пять минут на Google News, я обнаружил следующие сенсационные истории:

Плюс топики на форумах DZone, SlashDot, The Server Side. Плюс почтовые рассылки.

Всё это «выросло» из одного твита, «проверенного» «журналистами».

Некоторые наши потратили немало времени на выходных, чтобы донести до народа правильную информацию. Вчера Генрик (видимо, Henrik Stahl) опубликовал пост о намерениях Oracle относительно JVM, Dalibor использовал его для написания своего поста Extra! Extra! Oracle Premium JVM! Read all about it!. Но гораздо легче распространять дезинформацию, нежели корректировать её. Взгляните на список новостей выше и скажите, сколько там правильных новостей.

Мораль? Для меня их две:

1) Журналисты (блоггеры, твитеры) должны всегда помнить слова Graeme Philipson (@robilad):
«Журналисты должны быть осторожны в нынешний онлайн-век постоянной дезинформации»

2)Читатели должны со скептицизмом читать новости.

Конечно, звучит как мамины наставления, но никогда не бывает лишним напомнить ещё раз.

Перед тем, как мы вернёмся к тематическим топикам, вспомню о подобной истерии относительно InnoDB и MySQL. Комментарии излишни — читайте Get The Facts: MySQL Licensing and Pricing.

PS. Я ни в чем не обвиняю Майкла, с его твитом всё было ок.
