# WikiSearch
Парсер необходимый для скачивания корпуса докуметов.

#### Установка расширений
```
pip install requests
```

#### Начальная настройка
В случае необходимости, заменить относительные пути на абсолютные
```
.\venv\Scripts\python.exe .\HabrParser.py
```
