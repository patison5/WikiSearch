Shop2CD — создание сайтов-сателлитов PHPShop и копий каталога на СD

Блог компании PHPShop Software
Компания PHPShop Software — разработчик CMS для интернет-магазинов представила новую утилиту, которая позволяет с легкостью создавать копии каталога интернет-магазина для записи на компакт-диск или для создания сайтов-сателлитов.

Утилита Shop2CD — инструмент продвижения интернет-магазина, который можно использовать для создания сайтов-сателлитов или в рамках партнерских программ, когда каталог магазина размещается на сайте партнера.

Программа создает html-страницы с описанием и изображением товаров в интернет-магазине, при этом не требуетcя дополнительная лицензия. Генерация CD проходит в on-line режиме и позволяет создать полную копию вашего каталога товаров. При просмотре каталога товаров на CD покупатель может сразу же перейти на сайт или добавить товар в корзину с последующим оформлением заказа в интернет-магазине.
Утилита поддерживает различные темы оформления и легко настраивается под персональный дизайн.







Shop2CD входит в пакет утилит PHPShop EasyControl, бесплатно скачать который можно на официальном сайте.
