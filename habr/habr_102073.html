FVD Suite — Качает Всё! Ну или почти всё…

Блог компании FVD Media
Дорогие Друзья!
Рады вас приветствовать на нашем блоге, который мы надеемся займет достойное место на хабракарте.

Наша компания занимается разработкой программного обеспечения для работы с медиаконтентом. Об выходе новой версии одного из наших продуктов, а именно FVD Suite и пойдет речь в этом посте.

FVD Suite это возможность скачивать Ваше любимое видео и кодировать в нужный формат в несколько кликов мышью.




Программа полностью бесплатная и имеет награду Editor’s Pick от Brothersoft.com

Поддерживается множество различных сервисов, включая самые популярные YouTube, Metacafe, Megavideo…
И главное нововведение — автоматический поиск активных потоков (то что Вы просматриваете) и их загрузка.

Интерфейс достаточно прост, главное окно разбито на вкладки. Но обо всем по порядку.

Скачать аудио/видео


Здесь Вы можете скачивать контент с помощью веб ресурсов FVD.
Список поддерживаемых сайтов можно получить, нажав кнопку List of supported sites.
Просто вставляете URL в поле, и нажимаете кнопку Add.


также можно указать формат, в который будет отконвертировано видео по окончании загрузки.

На данный момент поддерживаются следующие форматы:
AVI, FLV, WMV, MOV, MP4, PSP, 3GP, SWF.
MP3, WAV, OGG, MP4, FLAC.

Есть функции импорта и экспорта в M3U плейлисты.

Потоковое медиа и RTMP потоки


С недавнего времени программа может находить активные потоки аудио/видео по протоколам HTTP и RTMP с помощью библиотеки WinPcap.
Все что Вам нужно, это выбрать нужный сетевой интерфейс из списка, или воспользоваться авто-поиском.



Программа умеет работать с видео, аудио, и флеш данными.
Список поддерживаемых MIME типов представлен ниже
  • video/flv
  • video/x-flv
  • video/mp4
  • video/x-m4v
  • video/3gpp
  • video/quicktime
  • video/ogg
  • audio/mp4a-latm
  • audio/mp4
  • audio/ogg
  • audio/x-mp3
  • audio/mpeg
  • application/x-shockwave-flash


Конвертер


Конвертер это то, с чего начиналось развитие программы.
Кодировать файлы можно как по отдельности, так и целым списком.



Из настроек можно выбрать кодек аудио/видео, битрейт, разрешение, частоту дискретизации.

Поиск видео


Не обошлось и без функции поиска видео. Используется Google Custom Search.
Поиск ведется только по тем сайтам, которые поддерживаются FVD Suite.

Интеграция с браузерами


Чтобы работать с программой было еще проще, мы разработали расширения к браузерам (пока для Firefox и Internet Explorer).


Кнопка Download, автоматически отправит нужный URL на скачку.
Еще стоит упомянуть бонусные функции
  • Make short url — сокращалка ссылок
  • Share url — поделиться ссылкой с помощью AddThis
  • JavaScript — включение и отключение JavaScript во время серфинга
  • Check IP — проверка IP
  • SpeedTest — проверка скорости соединения
  • WhoIS.


Техническая сторона


В данный момент софт работает на платформе Windows. Однако планируется выпустить версии для MacOS X и систем семейства Linux.
Функции определения потоков реализованы с использованием библиотек WinPcap.
В роли энкодера выступает всем известный FFmpeg.



Для загрузки по протоколу HTTP используется cURL, а для RTMP — проект flvstreamer.
Используется модульная архитектура, где для каждой функции свой бинарник. А связь происходит через стандартный ввод-вывод. Таким образом процесс портирования займет меньше времени, например в MacOS cURL уже встроен.

Исходники, копии LGPL/GPL и инструкции по сборке FFmpeg и FLVStreamer, Вы можете найти на нашем сайте.
