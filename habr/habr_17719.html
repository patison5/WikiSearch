Correo: почтовый клиент для Macintosh на базе кода Thunderbird с нативным интерфейсом

IT-компании
Nick Kreeger, один из контрибьютеров проекта Camino, выпустил на днях очередную версию своего детища — почтового клиента на базе кода Thunderbird с нативным интерфесом Mac OS X под названием Correo (что в переводе с испанского означает почта). Пока что почтовик обладает лишь самыми базовыми возможностями, однако в планах автора разработка API для написания плагинов, табы, поддержка адресной книги и другие улучшения.
