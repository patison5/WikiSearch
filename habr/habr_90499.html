Вы в восторге от панели задач, но работаете в Windows XP?

Разработка под Windows *
Несмотря на бурный рост популярности Windows 7 на сегодняшний день остается большое число пользователей Windows XP. При этом зачастую пользователи Windows XP ощущают преимущества Windows 7, но по каким-то причинам не могут перейти на новую операционную систему.

Как известно, одной из наиболее интересных возможностей Windows 7 является панель задач (Super Bar). Поэтому энтузиасты разработали программное обеспечение, позволяющее превратить панель задач Windows XP в панель задач Windows 7. Итак, встречайте – приложение ViGlance позволяет сделать панель задач Windows XP очень похожей на ту, что мы видим в Windows 7.



Для демонстрации того, как будет выглядеть панель задач в Windows XP проще посмотреть ролик, записанный автором.

Так что если вы вынуждены до сих пор пользоваться Windows XP – это не повод расстраиваться! Обновляйтесь и работайте еще более эффективно в т.ч. на Windows XP.
