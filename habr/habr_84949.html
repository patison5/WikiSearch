Skype прекращает поддержку Skype Lite и Skype для Windows Mobile

Мессенджеры *
Последние изменения в политике Skype, касающийся поддержки мобильных устройств происходят тихо и незаметно. Без громких пресс-релизов и новостей на внутреннем блоге. Так на днях в открытом доступе появилась страничка финального Skype for Symbian, который до настоящего момента фигурировал исключительно в качестве пробной бета версии.

А сегодня на страничке Skype для мобильного телефона появилась скромная строчка: Skype Lite и Skype для Windows Mobile больше не поддерживаются. Для получения подробной информации нажмите здесь.

К сожалению, по ссылке на подробную информацию никаких подробностей не представлено, так как ссылка ведет на страницу функции Skype To Go.

Впрочем, мне все таки удалось найти объяснение происходящему. Согласно ответу из FAQ секции сайта:

"Почему я не могу загрузить Skype Lite или Skype для Windows Mobile?

К сожалению, Skype Lite — версия Skype для мобильного телефона — и Skype для Windows Mobile больше не доступны для скачивания с нашего сайта.

Мы решили прекратить поддержку Lite Skype и Skype для Windows Mobile, потому что мы хотим предложить нашим клиентам новые улучшенные продукты для мобильного использования, такие как Skype для iPhone, которая оказалась столь популярной, и доступный теперь Skype для Symbian телефонов.

Нашей целью является обеспечение эффективной работы пользователей, предоставив возможность свободно общаться Skype-to-Skype и звонить на обычные телефоны по низким тарифам так же легко, как это происходит на настольном компьютере.
Мы чувствовали, что Skype Lite и Skype для Windows Mobile не позволяли полноценно использовать все возможности Skype.

Могу ли я использовать Skype Lite, если она уже установлена?
Да. Пользователи, которые уже установили Skype Lite могут продолжать использовать ее в течение 2010 года.

Могу ли я использовать Skype для Windows Mobile, если она уже установлена?
Да. Пользователи, которые уже установили Skype для Windows Mobile, конечно, могут продолжать использовать его."

Несмотря на это, сами странички продуктов пока еще доступны: Skype для Windows Mobile, Skype Lite.
