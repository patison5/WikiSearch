Хитрости Nokia 5800 и N97

Чулан
Перевод статьи "The ultimate guide to Nokia 5800 and N97 tweaks"

Общие советы

  1. В режиме набора номера двухсекундное нажатие на # приводит к переключению режимов «Без звука» и «Обычный»; нажатие и удержание 0 откроет веб-браузер; нажатие и удержание 1 откроет голосовую почту.
  2. Под глазком передней камеры в правом верхнем углу расположена кнопка быстро запуска «Музыкального проигрывателя», «Галереи изображений и видеоклипов», «Интерактивного обмена», «Видеоцентра» и Веб-закладок.
  3. Нажатие и удержание зеленой кнопки вызова активирует режим голосовых команд.
  4. Сгруппируйте ваши sms-сообщения, также как это позволяет сделать Outlook. Для это пройдите по пунктам меню: Сообщения -> Функции -> Параметры -> Другие -> Показыв. сбщ в группах -> Да.
  5. Для настройки уведомления о получении сообщения и пропущенного звонка пройдите по пунктам меню: Параметры -> Телефон -> Световые индикаторы
  6. Если нажатия на экран стали не аккуратны, вы должны откалибровать сенсорный экран — Параметры -> Телефон -> Сенсорный ввод -> Калибр. сенсор. экрана
  7. Включение функций контроля:
    • Поворачивайте телефон горизонтально или вертикально;
    • Для выключения звонка при входящем вызове переверните телефон «лицом» вниз;
    • Для выключения будильника переверните телефон «лицом» вниз.
  8. Для включения/выключения 3D эфектов тем и эффектов трансформации(значительно «тормозит» телефон) — Параметры -> Персональные -> Темы -> Обычные -> Функции -> Эффекты темы -> Вкл/Выкл
  9. Быстрая клавиша выхода из приложения: нажмите и удерживайте среднюю белую кнопку, затем нажмите и удерживайте значек программы из которой хотите выйти, выберите открыть/выйти в выпадающем меню(лучше для этих целей использовать программу handy taskman)
  10. Во время использования ручного ввода, прочерк слева на право вставит пробел, справа налево удалит символ.
  11. Organize your menu by Menu -> option -> organize. You could drag and drop the location of the application you wish to relocate.
  12. Для обновления прошивки Nokia 5800 наберите *#0000#, отобразится информация о установленном ПО. Теперь выберите в меню Проверить обновление. Новейшее обновление будет отправлено на ваш телефон от nokia, Скачайте и установите обновление(потребуется GPRS или Wi-Fi соединение)
  13. Нажмите на часы, чтобы открыть меню и настроить будильник.
  14. Нажмите на индикатор режима Обычный или дату и в выпадающем меню выберите режим, это почти тоже самое, если нажать на кнопку выключения на всех телефонах с S60, но уже без пункта «Открыть календарь»
  15. Тоже что и первый пункт, но другими словами.
  16. Нажимая на пункт в списке To do или в календаре вы немедленно перейдете к этому пункту
  17. Можно нажимать на иконки активности в верхнем левом углу(около индикатора батареи)(у меня на Nokia 5800 эти иконки появляются в правом верхнем углу -прим. переводчика)
  18. Долгое нажатие на зеленую кнопку вызова откроет голосовую почту
  19. Долгое нажатие на центральную кнопку(аналогично нажатию на кнопку Меню(кнопка инь-янь на некоторых S60)) откроет меню всех активных приложений.
  20. Для создания резервной копии(backup) данных телефона выберите Приложения -> Диспетчер файлов -> Резервная копия. Можно настроить автоматический ежедневный/ежемесячный backup.
  21. Во время серфинга в Интернет, если надо увеличить страницу, просто дважды нажмите на область которую хотите увеличить. Обычно достаточно простого перетаскивания страницы до нужного места, но также можно воспользоваться полосами прокрутки. Эта прекрасная возможность дает вам представление о размере страницы.
  22. Для переключения между режимами «Обычный» и «Без звука» нажмите и удерживайте # в режиме набора номера.
  23. Для просмотра списка всех открытых приложение нажмите и удерживайте кнопку меню.
  24. Обучение рукописному вводу — Параметры -> Телефон -> Сенсорный ввод -> Обучение рукописному вводу
  25. Для жесткой перезагрузки телефона, просто выключите телефон, одновременно нажав красную, зеленую кнопки и кнопку включения камеры(у меня на моем Nokia 5800 не получилось -прим. переводчика).
  26. Get the Red One and partner it with the Red Nokia BH-503 Stereo Bluetooth to get an awesome wireless sonic experience.

Советы по энергосбережению

  1. Отключите постоянное сканирование беспроводных сетей и bluetooth, если не используете их.
  2. Также не пользуйтесь 3G, выбрав режим GSM вместо «Dual mode» — Параметры -> Связь -> Сеть -> Режим сети -> GSM
  3. Понизьте яркость дисплея
  4. Всегда закрывайте приложение, если больше не пользуйтесь им.

Советы по сохранению памяти

  1. Перезагрузите телефон, если памяти осталось меньше чем 30Мб, либо сделать мягкую перезагрузку(в режиме набора номера наберите 5**(у меня на моем Nokia 5800 не получилось -прим. переводчика)) или перезагрузите с помощью кнопки выключения телефона.
  2. Используйте Handy Taskman для закрытия приложений или «убиения» не используемых приложений

Советы по использованию камеры

  1. Шумы можно уменьшить, если установить чувствительность ISO низкой вместо автоматической
  2. Во время просмотра изображений можно листать влево и вправо, также как и в iphone.

Видео

Для конвертирования видео в наилучшем качестве для Nokia 5800, следует использовать формат mp4 как показано ниже(все в формате 16:9):
  1. Xvid @ 678kbps разрешение 480×272
  2. Xvid @794kpbs разрешение 640×352
  3. H.264 @ 512kbps разрешение 400×224
  4. H.264 @ 550kbps разрешение 400×224 (рекомендуется).

Музыка

  1. Если вы не можете найти композиции после копирования их с компьютера на карту памяти — обновите библиотеку в музыкальном плеере Функции -> Обновить фонотеку

Приложения

  1. Всегда устанавливайте темы на карту телефона. Потому что некоторые темы не работают(Например, Christmas tree) не из памяти телефона.
  2. Корректно удаляйте приложения — Параметры -> Диспетчер приложений -> Установка приложений -> выберите приложение, которое нужно удалить
  3. Включите 3D-эффекты в вашей теме — Параметры -> Персональные -> Темы -> Обычные -> Функции -> Эффекты темы -> Вкл/Выкл

Некоторые коды которые могут пригодится на любом Nokia 5800 XpressMusic или N97

  1. *#06# покажет IMEI номер вашего устройства
  2. *#2820# покажет адрес вашего blutooth-устройства
  3. *#7380# восстановит настройки по умолчанию с паролем по умолчанию 12345
  4. *#92702689# покажет Life Timer(общее время разговоров)
  5. *#62209526# покажет MAC-адрес используемого Wi-Fi
  6. *#7370# — мягкий сброс телефона с паролем по умолчанию 12345
