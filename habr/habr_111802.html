Выпущены финальные версии ASP.NET MVC 3, WebMatrix, IIS Express, SQL Server Compact 4.0, Orchard 1.0 и Nuget

.NET *


Microsoft выпускает финальные версии своих инструментов веб-разработки. Для загрузки и установки доступны фреймворк ASP.NET MVC 3 (вместе с исходными кодами под опенсорс-лицензией) и среда разработки WebMatrix 1.0.

ASP.NET MVC 3 привносит следующие нововведения по сравнению со второй версией:
  • внутренняя интеграция с IoC-контейнерами и повсеместная поддержка Dependency Injection;
  • добавлен новый движок представлений Razor ViewEngine;
  • расширение механизма валидации: новые атрибуты, поддержка IValidatableObject, исключение параметров при валидации запросов;
  • поддержка динамического типа данных (зависимость от .NET 4.0);
  • глобальные фильтры;
  • встроенная поддержка JSON при работе с параметрами действий;
  • ненавязчивый JavaScript;
  • бессессионные контроллеры;
  • кэширование вложенных действий.
Более подробно обо всех этих нововведениях можно узнать из следующих статей: первая часть, вторая часть, третья часть, четвертая часть.

Нужно сказать, что финальная версия ASP.NET MVC 3 RTM отличается от предыдущей версии RC2 (выпущенной в декабре 2010 года) только исправлением пары ошибок.

Загрузить ASP.NET MVC 3 RTM можно по следующей ссылке.

Для удобства разработчикам предложена утилита миграции проектов ASP.NET MVC 2 на ASP.NET MVC 3, которую можно загрузить по следующему адресу.

Вместе с ASP.NET MVC 3 распространяется пакетный менеджер для .NET-проектов под названием Nuget. Nuget — это совершенно новый проект, про который вы можете узнать из этой статьи. Nuget — очень полезен, крайне рекомендуется к ознакомлению. Для него создано уже несколько сотен пакетов.

Кроме того, сегодня выпущена финальная версия опенсорс-проекта Orchard CMS 1.0, которая построена на базе ASP.NET MVC 3. Загрузить проект можно из Web Platform Installer или по следующей ссылке.

Другим релизом сегодня является выпуск среды разработки WebMatrix. WebMatrix – это бесплатная среда разработки для создания, оптимизации и размещения веб-проектов на удаленном хостинге. WebMatrix предлагает инструменты по работе с базами данных и кодом на ASP.NET и PHP. В галерее среды разработки есть возможность выбрать готовые проекты из числа известных: Joomla, Drupal, WordPress, DotNetNuke, BlogEngine.NET, Umbraco CMS, SrewTurn Wiki и других.

  

Немного подробнее об этом проекте можно прочитать тут. Официальная страница проекта доступна по этому адресу. Установить WebMatrix можно по следующей ссылке.

Отдельно доступны для загрузки следующие инструменты, которые являются составными частями WebMatrix, но могут использоваться отдельно:
  • фреймворк ASP.NET Web Pages (с синтаксисом Razor);
  • Internet Information Services (IIS) 7.5 Express. Описание IIS Express можно почитать тут;
  • SQL Server Compact 4.0 (ссылка на x64, ссылка на x86). Что в нем нового и интересного можно узнать из этой заметки.
