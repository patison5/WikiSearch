Испытание сервисов для создания коротких ссылок

Чулан
Зачем был написан топик?

Испытав в очередной раз необходимость воспользоваться TinyUrl, я наткнулся на угрожающее сообщение, что либо сайт находится в блэк-листе, либо мой IP забанен. Попробовав ввести другой адрес и добившись результата, я пришёл к неутешительному выводу, что такую фигню выдаёт ссылка на файл, лежащий на яндексовом хостинге. Пара минут поиска — и вот ссылка уже благополучно обрезана в MooURL. Откуда я взял MooURL? А вот наобум выбрал. В своём блоге, в записи от 29.06.2008, Артём Беляйкин опубликовал список из 68 сервисов, которые предложил для уменьшения размера длинных ссылок. Правда, с упором на маскировку партнёрских ссылок, а я упираюсь в пользователя :)

Поскольку выбирать наобум — это путь, всё-таки, немного не наш, было решено просмотреть все ссылки и оформить о них первое впечатление в виде экспресс-обзора. По ходу дела ещё и выяснилось, что для Cети полгода — это уже Срок, и часть из этих сайтов уже не работает.

Зачем это хабрасообществу?

Для пользы. И чтобы не копаться в этом самим, когда понадобится. Использовано может быть по разному поводу, например: записывать на бумажку такой адрес удобнее; по смс скидывать удобнее; в социальных сетях (когда стоит запрет на публикацию адреса) публиковать удобнее. Это если не брать в расчёт использование в целях редиректа и прочие варианты.

Что же было сделано?

Была проделана работа. Немаленькая. По тестированию всех ссылок. Для тестирования возможностей использовался адрес сайта «http://habrahabr.ru/». В реальности он, разумеется, удобен и сам по себе, но у нас цель не похвастаться, насколько сокращаются длинные адреса, а сразу показать, насколько это лучше коротких же имён, если они плохо выглядят/запоминаются/etc. Хоть к Хабру это и не относится :) Там, где можно задавать «предпочтения» (да, звучит ужасно), то есть ссылки, включающие заданную пользователем комбинацию символов, использовалось предпочтение «habr» или похожее. Все оценки субъективны и основаны на моём общем восприятии или невосприятии сайта. Основной интерфейс ссылок — английский (хотя встречаются немецкий, французский, португальский и т.д.).

Надеюсь, этот обзор поможет кому-то выбрать свой ресурс необходимой функциональности. Итак, поехали.
 
 

Работают после регистрации


DigBig
Выдаётся адрес вида digbig.com/3bbd. Есть букмарклеты, статистика.
lnk.in
Как написано в описании, сайт позволяет создавать ссылки обычным путём, с заданными комбинациями, в пакетном режиме (когда заменяются все валидные ссылки на страницах), а также размещать свою кнопку преобразования адреса. Есть статистика и защита ссылок паролем.
NanoRef
Минималистичный интерфейс, позиционируются как заменители адресов для писем и чатов. Есть возможность задания предпочтений, когда создаётся адрес вида abcdef.nanoref.com.
SHurl
Говорят только о наличии статистики и отслеживании ссылок.
Sitelutions
Целая система для работы с доменами.
 
 

Работают после регистрации (после выбора домена)


Ne1
Сайт работает по необычной схеме. Сначала вы выбираете один из трёх доменов, запрашиваете его доступность, потом заполняете персональные данные — имя, e-mail, пароль, адрес сайта и даже его категорию. Для Хабра мог бы быть создан адрес habr.r8.org, если бы всё не было так муторно. 4/10.
Shim
Механизм работы — как у Ne1. Выбор из семи доменов. Большой регистрационный бланк. Для Хабра получилось бы что-то вроде habr.2cd.net. 4/10.
Ulimit
Механизм работы — как у Ne1. Выбор из 26 доменов. Тоже большой регистрационный бланк. Для Хабра получилось бы что-то вроде habr.it.st. 5/10.
 
 

Работают


Beam.to
Для преобразования существует кнопка «Add New URL», далее обязательно нужно указать комбинацию-предпочтение, сайт и почтовый ящик. Для Хабра было бы что-то вроде «beam.to/habr» (не преобразовывал). Система учётных записей. Позиционируют себя как сервис для улучшения вида имени вашего сайта, «выглядящего плохо». 3/10.
CanURL
Для Хабра — canurl.com/ab5ys. Симпатично. Говорят, что оптимизированы для мобильных устройств. 7/10.
Dwarfurl
Для Хабра — dwarfurl.com/d2a70. Позволяет опционально задавать пароль для просмотра количества переходов по вашей ссылке, имеет опцию «может ли ссылка публично показываться». Сразу же генерирует HTML-код для вставки ссылки. 8/10.
Get Shorty
Для Хабра — get-shorty.com/shorty/habrahabr. Сайт, претендующий на стильность, но создающий не самые удобные ссылки. Здесь, кстати, нам предлагают ещё и что-то скачать для автономной работы. Ключевое слово задаётся разве только для дальнейшей настройки адреса. 4/10.
Is.gd
Для Хабра — is.gd/dCW. Хвастается, насколько укоротил адрес. Если включён flash, адрес автоматически копируется в буфер. На странице инструкций есть ссылки на соответствующий букмарклет, плагин для Firefox и виджет для Mac OS X. Добавление в конец образованного адреса дефиса (is.gd/dCW-) отправляет на страницу информации о ссылке. 6/10.
Nutshell URL
Для Хабра — nutshellurl.com/3t1. На главную добавлено поле Google Custom Search. Всё просто и понятно. Введя nutshell-адрес в специальное поле, можно узнать, куда он ведёт. Есть некое подобие маскота — толпа грецких орехов :) 7/10.
Redirx
Для Хабра — redirx.com/?sg3o. Образец чуть приукрашенного минимализма. На главной — пара слов и поле для ввода адреса. Под логотипом обнаруживается ссылка на FAQ. 6/10.
RubyURL
Для Хабра — rubyurl.com/N8Yu. Новообразованная ссылка по выбору может быть сразу скопирована в буфер. Есть свои букмарклет и API (с примерами). 8/10.
Shorl
Для Хабра — shorl.com/brygynihyfove. После нажатия на кнопку выдаёт устрашающе выглядящий «короткий» адрес («шорл») и устрашающий же пароль к нему (для пользования статистикой). Есть букмарклет. 4/10.
Shoter Link
Минимализм с умом: для Хабра сразу же честно признался, что адрес меньше не сделает. В качестве примера приведён shorterlink.com/?ABC123. 5/10.
ShortURL
Для Хабра — alturl.com/53o. Всё пестро, обилие деталей сбивает с толку. Система учётных записей. Есть букмарклет. Задание комбинаций (и ещё пара полезных возможностей) вроде как тоже есть, но расписано очень сумбурно. 5/10.
Short URL
Для Хабра — url.lotpatrol.com/?x=1855. Говорят, что преобразованием почтовых адресов в свои адреса защищают от web-spider'ов. 5/10.
SimURL
Для Хабра — simurl.com/matruw. Отслеживать статистику можно после регистрации. После создания ссылки тут же пытается загрузить заданный сайт в свой фрейм. 6/10.
Smallr
Для Хабра — smallr.com/921. Симпатично и минималистично. Есть букмарклет и API. 8/10.
Tight URL
Неказистый интерфейс из мешанины текста. Для Хабра не выдал ничего, извинившись. Скорее всего, как и Shoter Link, он просто приходит к выводу, что адрес меньше не сделает. Но подробности от пользователя утаивает. Есть букмарклет. 4/10.
URL.co.uk
Для Хабра — url.co.uk/vrsdn. Сразу же даётся HTML-код и код для вставки на форумах. Симпатичный минималистичный интерфейс. 8/10.
URl.ie
Для ссылки генерируется код для форумов, QR Code и показаны возможные действия со ссылкой (пересылка, добавление в закладки / социальные сети). Лёгкий интерфейс, мягкие цвета, листок клевера на фоне и интересная функциональность. Что ещё надо? Букмарклет? Он есть. 9/10.
WapURL
Для Хабра — wapurl.co.uk/?M0FC3EJ. Статистика по адресам, букмарклеты (FF, IE), API. Скоро обещают ввести регистрацию. 5/10.
X.se
Для Хабра — www. x.se/dxf9 (именно так, с «www», иначе выдаёт «No DNS records»). Сайт на шведском, в коем я не силён, а потому мог что-то пропустить. Работает шустро и без перезагрузки выдаёт новый адрес. Там же предлагается зарегистрировать .se- или .asia-домен. Всё. 5/10.
YATUC
Для Хабра — yatuc.com/uub. YATUC — «Yet Another Tiny Url Creator». Статистика, защита ссылок, задание пароля. Редирект с учётом добавленных уровней. Немного разъезжающийся интерфейс на английском и немецком. 6/10.
Pi URL
Для Хабра — piurl.com/1FiL. Превью адреса, редирект с учётом уровня, API, букмарклет. Ключевые слова в адресе для его описания (добавляются пользователем при публикации ссылки). 6/10.
URLvi
Для Хабра — urlvi.be/mrshj. Ведётся статистика по адресу, пароль к ней задаётся системой, показывается сразу на странице преобразования, важно не просмотреть. Есть букмарклет. 5/10.
 
 

Работают с заданными комбинациями


Decent URL
Для Хабра с предпочтением — habrahabr.decenturl.com/habr. Без предпочтения — habrahabr.decenturl.com/half-decent-el3q. Т.о., для доменного имени второго уровня задаётся уровень сайта DecentUrl, а после слэша помещается сочетание-предпочтение (если было задано) или произвольное. Решение не самое удачное для простого сокращения адреса, но для плохого адреса сайта или для запоминания (а не записывания бессмысленных сочетаний) сайта может быть удобно. 6/10.
Doiop
Для Хабра с предпочтением — doiop.com/habr, без предпочтения — doiop.com/g39ct8. 6/10.
EasyURL
Для Хабра с предпочтением — atu.ca/habr. Без предпочтения были выданы два варианта — ecc.atu.ca и atu.ca/ecc. Позволяет задавать один из пяти доменов, на основе которых будет создана ссылка (самый короткий — atu.ca). На сайте также присутствует сервис закладок, регистрация по OpenID. 8/10.
EzURL
Для Хабра с предпочтением — ezurl.eu/habr, без предпочтения у меня уже не сработало. Задаётся пароль для удаления или правки ссылки в будущем. 5/10.
FhURL
Для Хабра — fhurl.com/c3598. Всё аскетично и понятно. В верхнем меню скрывалась неприметная ссылка «Custom Url» (задание предпочтений), по ней — варианты загрузки заданного сайта «Redirect» (обычный редирект; fhurl.com/habr) и «Iframe» (фрейм, можно задавать Description и Keywords; fhurl.com/habr2). 6/10.
Fly2
Для Хабра с предпочтением — fly2.ws/habr, без предпочтения — fly2.ws/m887dbl. AJAX-интерфейс без полной перезагрузки страницы. Под рукой — кнопка копирования образованной ссылки в буфер. Интеграция в Google Toolbar. Система учётных записей. Выглядит симпатично. Есть свой маскот — пчёлка. Говорят о себе, что в пять раз быстрее, чем TinyURL. 9/10.
Metamark
Для Хабра с предпочтением (называется здесь Nickname) — xrl.us/habrahabr («habr» уже был занят). Большой минус — если сайт уже был добавлен в базу, вам только предложат уже существующие ссылки (которые могут быть не самыми удачными). Есть сомнительная возможность добавлять к урлу «секрет» (некую свою дополнительную комбинацию символов). Есть букмарклет. 3/10.
Minilien
Французская версия Tiny Link (описание аналогичное).
MooURL
Для Хабра — moourl.com/oh8j1. Образованная ссылка сразу копируется в буфер. После создания короткой ссылки появляется возможность «Give your moourl a personal touch», под которой скрывается знакомое задание предпочтений (для Хабра создалось приятное moourl.com/habr), после чего работают обе ссылки. Выглядит симпатично. Есть свой маскот — коровка. Есть букмарклет. Кроме английской, есть португальская версия сайта. 8/10.
MyURL
Для Хабра с предпочтением — myurl.in/habr, без предпочтения — myurl.in/gdMVT. Есть букмарклет. Присутствует режим пакетного переименования, когда можно задать сразу несколько адресов, по одному на строку. Создание короткого адреса, анонимизация и защита адреса. На сайте есть возможность создания почтового ящика в один клик (на самом деле — в два) вида abcdef@myurl.in на 24 часа. 9/10.
Not Long
При задании обязательной комбинации-предпочтения вроде как создаётся адрес вида abcdef.notlong.com (увы, три попытки для Хабра закончились неудачей). Есть своя статистика, букмарклет и возможность использования API в Perl. 5/10.
SnipURL

Думаю, что нет смысла расписывать, что есть стандартная работа с редиректом, задание alias'ов, превью-урлы, кнопка на панель инструментов. Есть блэк-лист на круг адресов для преобразования. 8/10.

Итак, мы выкидываем из этого списка 17 неработающих адресов, и у нас остаётся 51 ресурс (+ ваши) для того, чтобы кромсать свои длинные и не очень ссылки. Разумеется, такие краткие тексты обзоров не годятся для немедленного выбора «своего» ресурса. Поэтому возьму на себя смелость кое-что из этого списка порекомендовать.

Если вам нужны системы-тяжеловесы с кучей наворотов, то стоит обратить внимание на EasyURL, Fly2, MooURL, MyURL, SnipURL, TinyURL, Zi.ma.
Если же по душе интерфейс в стиле «кликнул-получил», попробуйте B23, Clck (оба на русском!), Dwarfurl, Is.gd, Nutshell URL, RubyURL, Smallr, URL.co.uk, URLvi.
Для каких-то иных целей (например, жутко хочется иметь букмарклет или отслеживать статистику) — смотрите описание.

P.S. Поискав на Хабре, нашёл ещё Tr.im, симпатичный ресурс с заданием предпочтений для адреса и завязкой на твиттер.
P.P.S.
