Обзор SeaMonkey 2.0

Браузеры


После двух лет ожидания разработчики-энтузиасты выпустили финальную версию интернет-комбайна SeaMonkey 2.0.

Осторожно, трафик!



Что же представляет из себя этот зверь ?


— Встроенный почтовый клиент на основе Thunderbird
— Браузер, практически ничем не отличающийся от Firefox
— HTML-редактор
— IRС — клиент ChatZilla
— Адресная книга

Какие же плюсы и минусы у SeaMonkey по сравнению с отдельно взятыми Firefox, Thunderbird, ChatZilla ?

Плюсов несколько:

— Не надо устанавливать 3 программы по отдельности
— Удобная функция сворачивания панелей
— Интеграция трех программ в одну
— Скорость загрузки программы из «холодного» состояния выше, чем у Firefox

Минусов меньше, но они есть:

— Дополнения от Firefox не подойдут к SeaMonkey, а в списке дополнений на момент написания статьи было всего 2 аддона — FlashGot и NoScript. Список постепенно увеличивается.

— Не очень удачное расположения кнопки закрытия вкладки, приходится тянуться из одного края монитора в другой.

Приступим к обзору самого «комбайна»



Прежде всего нас ждет обновленный интерфейс. Тема не мозолит глаза



Функция сворачивания панелей — на скриншоте обе панели свернуты



Почтовый клиент ничем не отличается от Thunderbird, кроме как интеграцией в «комбайн» и измененным интерфейсом



Редактор HTML, хоть и без подсветки, но хорош. Позволяет просматривать набранный код, предварительно сохранив его. Позволяет сразу же заливать получившийся документ на сервер с помощью FTP. Тут же нашелся баг — проблема с кодировкой в титуле окна

Режим написания



Режим предпросмотра



Менеджер загрузок остался похожим на своего младшего собрата из SeaMonkey 1



Так же имеется отладчик JavaScript с подсветкой элементов кода



Неожиданным для меня был встроенный плагин переводчика. По умолчанию используется translate.ru



Поиск по странице осуществляется с помощью JavaScript



Тест Acid3. SeaMonkey набрал 93 балла, столько же набрал Firefox 3.5.3



Тест SunSpider. Firefox оказался быстрее, набрав 1092.8ms против 1606.6ms у SM



Вывод



Если честно, я ждал большего. На данный момент дополнений для SM не так много, но, думаю, что разработчики быстро заполнят эту нишу.
Хороший менеджер загрузок и средства отладки. Кушает оперативки меньше, чем Firefox. Для нетбуков с малым объемом оперативной памяти это будет огромным плюсом.
