ExtJs — обновления компонентов

Библиотека ExtJS/Sencha *
Довольно давно не заходил на сайт ExtJs. Что же там нового?
Обновился фреймворк до версии 3.0.3 и вместе с ним добавилось много новых фич. Обновился блог. Обновился раздел документации. Обновился раздел демо-приложений. Скорее всего обновился и визуальный редактор, но по недавним причинам, обычные смертные его не увидят. Только если подписчики ;)

Итак, в блоге появилась заметка Расширенное создание плагинов с ExtJS (Advanced Plugin Development with Ext JS). Поправьте, если перевел не верно. В кратце, в статье рассказывается о том, что такое плагины вообще, какие существуют способы расширения функционала плагинов, создание своего плагина с примерами кода и демо страницей.

В разделе демок так же большое пополнение:
  1. Таблицы (Ext.Grid)
    • Row Editor Grid — Редактируемая таблица, которая позволяет делать изменения прямо в ячейках. Данный пример так же демонстрирует использование Ext.chart package. (Просмотр)
    • Progress Bar Pager — Пример интеграции индикатора (ProgressBar) с Пейджинг панелью (Paging toolbar) с помощью пользовательских плагинов. (Просмотр)
    • Sliding Pager — Пример интеграции слайдера (Slider) с Пейджинг панелью (Paging toolbar) с помощью пользовательских плагинов. (Просмотр)
    • Grouping with Dynamic Summary — Расширенная группировка в таблице (Ext.grid), которая позволяет одновременно редактировать ячейку и включает в себя пользовательскую динамическую группировку. (Просмотр)
    • Grid Filtering (Обновлен) — Плагин для таблицы (Ext.grid), который дополняет меню фильтра новыми типами пользовательских данных. (Просмотр)
    • Buffered GridView — Ext.GridView оптимизированный для быстрого отображения только видимых строк. (Просмотр)
    • Editable Grid with Writable Store - Это хранилище (Store) использует JsonWriter для автоматической генерации CRUD запросов к серверу через стандартный HttpProxy.
    • RESTful Store with GridPanel and RowEditor - RESTful Store в купе с JsonWriter автоматически генерирует CRUD запросы к серверу.

  2. Табы(Ext.Tabs)
    • TabPanel Scroller Menu — Демонстрация меню открытых вкладок, которые не попадают в доступную видимую часть окна. (Просмотр)
    • Group Tabs- Демонстрация создания группировки табов, используя вертикальные табы. (Просмотр)

  3. Диаграммы (Ext.charts)
    • Charts — Демонстрация использования графиков 3 видов. (Просмотр)
    • Stacked Bar Chart — Демонстрация использования линейчатой диаграммы. (Просмотр)
    • Pie Chart — Пример работы с круглыми диаграммами. (Просмотр)
    • Reloaded Chart — Пример работы с диаграммами с их постоянным обновлением. (Просмотр)

  4. Деревья (Ext.tree)
    • Check Tree — Демонстрация дерева с использованием чекбоксов (checkbox) (Просмотр)

  5. Формы(Ext.form)
    • Spinner field — Демонстрация стандартного спинера (обычный textfield с двумя кноками по бокам для увеличения или уменьшения числа в textfield) (Просмотр)

  6. Меню(Ext.menu)
    • Toolbar Overflow — Автоматическое создание подменю (Ext.menu) для элементов панели инструментов, которые не попадают в видимую часть окна. (Просмотр)
    • Toolbar Button Groups — Группировка кнопок на панели инструментов. (Просмотр)


Среди обновленных элементов также замечены ListView, Panels, QuickTips
