Gigabit Switch Netgear GS504T

Старое железо
Доброго времени суток.
Раз уж заговорили о модемах, решил поделится своей старой железкой.

Однажды мне позвонил мой старый товарищ и сказал что у него завалялась какая-то компьютерная-сетевая штука… говорит: «Зайди глянь что это, а то лежит у нас в офисе уже 5й год, не знаем зачем оно нужно, только пыль собирает».




Это 4х портовый гигабитный свич (коммутатор?), Netgear GS504T, не управляемый.

Оригинальная коробка, со всем прилагающимся в комплекте: инструкция, гарантия производителя, крепления для стойки 19", кабель питания. Судя по инструкции и интернетам, 2000 года выпуска.

О возможностях и параметрах из инструкции:
Features
The Model GS504T switch has the following key features:
• Four full-duplex copper Gigabit ports with standard RJ-45 connectors
• Automatic address learning function to build the packet-forwarding
information table
The table contains up to 8,000 media access control (MAC) addresses (that is,
the switch can support networks with as many as 8,000 devices).
• Full-duplex mode to double throughput of point-to-point connections by letting
individual ports transmit and receive concurrently when the other end also sup-
ports full-duplex mode
• Wire-speed filtering and forwarding of the traffic with no delay
• Store-and-forward intelligent processing to minimize erroneous packets on the
network
• Easy Plug and Play installation with no software to configure,which saves time
and minimizes the potential for configuration errors

Network Protocol and Standards Compatibility
IEEE 802.3ab 1000BASE-T
IEEE 802.3u 100BASE-TX
IEEE 802.3x flow control

Interface

Physical Specifications

Weight:5.3 lb;2.3 kg

Все порты в рабочем состоянии, честно держат 1Гбит подключение.

*Красная симка для масштабов устройства




Но есть одно но… Заглянем внутрь.


Обилие вентиляторов, охлаждающих (управляющие?) микросхемы каждого порта, и ещё два охлаждающих весь корпус. Всё это вместе создаёт достаточно большой шум.

Не очень разбираюсь в устройствах свичей\коммутаторов, но логика подсказывает такую картинку. Сразу после сетевых разъёмов стоят трансформаторы для гальванической развязки с сетью (видимо чтоб не возникала разность потенциалов между устройствами); чипы с надписью «EtronTech» — оперативка, кешь память, по 3 на каждый порт; в центре «Galileo» — грубо говоря проц. Что под радиаторами и вентиляторами?
